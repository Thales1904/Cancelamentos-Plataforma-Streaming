# Análise de Cancelamentos em Plataforma de Streaming

Este projeto tem como objetivo investigar os **motivos por trás do cancelamento de clientes** em uma plataforma de streaming, utilizando análise de dados para gerar insights estratégicos.

Através do uso de ferramentas de análise e visualização, buscamos responder perguntas como:

* Por que os clientes estão cancelando?
* Existe algum padrão entre os usuários que cancelam?
* O tempo de uso influencia na decisão de sair?
* O preço ou plano impacta o churn?

---

## Metodologia

O projeto foi desenvolvido em duas etapas principais:

###  1. Tratamento e análise de dados

Utilizando a biblioteca **Pandas**, foram realizadas:

* Limpeza de dados (remoção de valores nulos, inconsistentes e inúteis)
* Transformações e organização das informações


###  2. Visualização dos dados

Com o uso da biblioteca **Plotly**, foram criados gráficos interativos que permitem:

* Identificar padrões de comportamento
* Explorar dados de forma dinâmica
* Facilitar a interpretação dos resultados

## Objetivo Final

Este projeto busca transformar dados brutos em **informações estratégicas**, auxiliando na tomada de decisões para:

* Reduzir o cancelamento de clientes (churn)
* Melhorar a retenção
* Entender melhor o comportamento dos usuários
